# assets/ — Recursos gráficos y de audio

## Sprites (SpaceRage Pack — Ravenmore, itch.io)

| Carpeta | Contenido |
|---------|-----------|
| `Player/` | Naves azul (`player_b_*`) y roja (`player_r_*`), 5 frames de inclinación c/u |
| `Enemies/` | 2 tipos × 3 colores (r/g/b) × 5 frames + minas animadas |
| `Explosions/` | 3 tamaños: grande (11 frames), mediana (9), pequeña (9) |
| `FX/` | Plasma (2 frames), vulcan (3), proton (3), exhaust/propulsor (5) |
| `Planets/` | 15 planetas de 128×128 px |
| `Asteroids/` | Sprites de asteroides en varias variantes |
| `Shadows/` | Sombras de naves (no usadas actualmente) |

## Fondo

| Archivo | Descripción |
|---------|-------------|
| `BG.png` | Fondo espacial tileable, se escala a la resolución del monitor |

## Audio

| Archivo | Uso |
|---------|-----|
| `musica_juego.mp3` | Música de fondo durante gameplay (loop) |
| `musica_boss.mp3` | Música durante boss fights (loop) |
| `musica_menu.mp3` | Música del menú principal (loop) |
| `disparo.mp3` | Disparo (en el juego se genera por síntesis; el archivo queda como respaldo) |
| `disparo_p2.mp3` | Disparo del jugador 2 (respaldo) |
| `explosion.mp3` | Explosión de enemigo normal |
| `explosion_grande.mp3` | Explosión del boss |
| `powerup.mp3` | Recoger power-up |
| `enemigo_muerto.mp3` | Enemigo destruido |
| `boss_alerta.mp3` | Alerta de aparición del boss |
| `nivel_up.mp3` | Subir de nivel |
| `compra.wav` | Compra exitosa en tienda |
| `sin_fondos.wav` | Fondos insuficientes en tienda |
| `menu_click.wav` | Navegación en menú/tienda |
| `oleada_completa.wav` | Oleada completada |
